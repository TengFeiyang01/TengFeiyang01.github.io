思路

思路

无向图的总度数为 2m2m2m ，这个图中，度为 xxx 的点有一个，度为 y+1y + 1y+1 的点有 xxx 个，度为 111 的点有 xyxyxy 个，那么有 2m==2x+2xy==x+x(y+1)+xy2m == 2x + 2xy == x + x(y + 1) + xy2m==2x+2xy==x+x(y+1)+xy ，解出 x,yx, yx,y 即可。

G. Hits Different

题目大意

给你一个 202320232023 行的金字塔，每一个位置 xxx 对应的权值为 x2x^2x2 ，问你将 xxx 位置击倒后能获得的权值是多少。

思路

预处理出来每一层的左右断点 L、RL、 RL、R 。记当前 xxx 所在位置距离当前层左侧的距离为 dldldl ，距离右侧边界的距离为 drdrdr，那么上一层倒塌的位置的左边界即为 max(L[i−1],L[i−1]+dl)max(L[i - 1], L[i - 1] + dl)max(L[i−1],L[i−1]+dl) ，右边界为 min(R[i−1],R[i−]−dr)min(R[i - 1], R[i - ] - dr)min(R[i−1],R[i−]−dr) ，依次统计即可（也可以用平方和公式加速）。

H. Don’t Blame Me

题目大意

给你一个长度为 n(1≤n≤2∗105)n(1 \le n \le 2*10^5)n(1≤n≤2∗105) 的数组 aaa ，数组中每个元素的范围为 1≤ai≤631 \le a_i \le 631≤ai​≤63 ，给你一个数字 k (0≤k≤6)k\ (0 \le k \le 6)k (0≤k≤6) ，问你 aaa 的所有非空子序列中，所有元素按位 ANDANDAND 后的值的二进制中包含 kkk 个 111 的子序列的个数。

思路

比较常见的 DPDPDP ，我们定义 dp[i][j]dp[i][j]dp[i][j] 表示当前处理到第 iii 个数，且前 iii 个数中按位 ANDANDAND 的结果为 jjj 的子序列的个数，每一次转移为：

dp[i][j]=dp[i−1][j]dp[i][j] = dp[i - 1][j]dp[i][j]=dp[i−1][j] （不包含当前的 a[i−1]a[i - 1]a[i−1] ）

dp[i][j&a[i−1]]=dp[i][j&a[i−1]]+dp[i−1][j]dp[i][j \& a[i - 1]] = dp[i][j \&a[i - 1]] + dp[i - 1][j]dp[i][j&a[i−1]]=dp[i][j&a[i−1]]+dp[i−1][j] （包含当前的 a[i−1]a[i - 1]a[i−1] ）
