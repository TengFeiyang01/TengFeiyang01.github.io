A. Divisible Array

题目大意

思路

考虑 ∑1n\sum_{1}^{n}∑1n​ 求和， (1+n)⋅n2\frac{(1 + n) \cdot n }{2}2(1+n)⋅n​ 这个式子在 nnn 为偶数的时候是一定能被 nnn 整除的，当 nnn 是奇数时，它的二倍是能够被 nnn 整除的，因此 aia_iai​ 的每一项都设为 2⋅i2\cdot i2⋅i 即可。

D. Range Sorting

题目大意

给你一个长度为 nnn 的数组 aaa ，你每次可以对一个区间 [l,r][l, r][l,r] 排序，花费的代价为 r−lr-lr−l ，问你对于 aaa 的所有子数组排序的代价和是多少。

EASY版本( 1≤n≤5⋅1031\le n\le 5\cdot 10^31≤n≤5⋅103 )

思路

枚举子数组的左端点，遍历右端点，使用单调栈维护后缀的最大值，栈中存放 (mx,cost)(mx, cost)(mx,cost) 表示 （当前后缀最大值，当前后缀最大值所在的区间排序所需的代价），我们每加入一个点，如果当前点大于之前的所有点，那么该点可以自成一段。

每次用当前值将大于当前值的元素弹出。（如果 a[j]a[j]a[j] 左侧有一个比它更大的数 xxx ，那么至少要把 a[j]a[j]a[j] 换到 xxx 的左侧。假设原来的代价 costcostcost ，则当前代价为 cost+1cost + 1cost+1 。这相当于用若干次弹栈将若干区间合并为一个区间。

E. Palindrome Partition

题目大意

给定字符串 sss ，定义美丽字符串为偶回文串，或者由若干偶回文串组合，问 sss 中有多少个子串是美丽字符串。

思路

有个结论，对于任意满足条件的串，总是可以贪心的拆成若干个满足条件小串，证明详见灵茶山艾府的这个题解：（https://leetcode.cn/problems/longest-chunked-palindrome-decomposition/solutions/2221544/tu-jie-tan-xin-zuo-fa-yi-tu-miao-dong-py-huik/）。

因此我们定义 dpidp_idpi​ 为以 iii 结尾的满足题意的美丽字符串的数量，那么记 iii 之前的一个最短的回文串的长度为 ttt ，那么有 dp[i]=dp[i−t]+1dp[i] = dp[i-t] + 1dp[i]=dp[i−t]+1 ，那么问题就变成如何找这个的 i−ti-ti−t 。我们可以使用 manachermanachermanacher 计算出当前位置 xxx 的回文半径，每次将 xxx 放入栈中， 如果 x+p[x]<ix+p[x] < ix+p[x]<i 那么直接将 xxx 出栈即可，否则我们可以通过回文串的右端点以及回文中心找到回文串的左端点 lll ，那么 dp[i]=dp[l−1]+1dp[i] = dp[l-1] + 1dp[i]=dp[l−1]+1 ，最后累加起来即可。

由于我们是从小到大遍历的，因而栈内的回文中心的下标都是递增的，即 xxx 是第一个满足 p[x]+x≥ip[x] + x \ge ip[x]+x≥i 的回文中心，那么 i−xi-xi−x 一定是最小的，那么回文串的左端点 2∗x−i2*x-i2∗x−i 一定也是最大的，这就满足了我们要找的 ttt 是 iii 之前最短回文串的起点的条件。
